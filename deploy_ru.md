# Как разместить этот сайт на GitHub Pages бесплатно

Вы создали красивый сайт-визитку для SigmaVPN! Теперь давайте разместим его в интернете, чтобы вы могли скидывать ссылку другим людям. GitHub предоставляет бесплатный хостинг для статических сайтов через функцию **GitHub Pages**.

## Шаг 1: Создание репозитория на GitHub
1. Перейдите на [GitHub.com](https://github.com) и войдите в свой аккаунт.
2. В правом верхнем углу нажмите на иконку **"+"** и выберите **"New repository"**.
3. В поле **Repository name** введите `sigmavpn` (или любое другое название без пробелов).
4. Убедитесь, что выбран вариант **Public** (Публичный), иначе GitHub Pages не будет бесплатным.
5. Нажмите кнопку **Create repository** в самом низу.

## Шаг 2: Загрузка файлов
1. На открывшейся странице репозитория найдите ссылку **"uploading an existing file"** (чуть ниже блока с кодом).
2. Откройте папку `SigmaVPN-Promo` на вашем рабочем столе.
3. Выделите файлы `index.html`, `styles.css` и `script.js` (можете также закинуть эту инструкцию `deploy.md`, это не повредит).
4. Перетащите их мышкой в окно браузера на GitHub.
5. Подождите, пока файлы загрузятся. Затем прокрутите страницу вниз и нажмите зелёную кнопку **Commit changes**.

## Шаг 3: Включение GitHub Pages
Всё почти готово! Теперь включим сам сайт:
1. В вашем новом репозитории на GitHub нажмите на вкладку **Settings** (Настройки) в верхнем меню.
2. В левом боковом меню выберите раздел **Pages**.
3. В блоке **Build and deployment** найдите выпадающий список **Source** и убедитесь, что там выбрано *Deploy from a branch*.
4. В разделе **Branch** (Ветка) нажмите на выпадающий список (там написано *None*) и выберите **main** (или master).
5. Нажмите кнопку **Save**.

## Шаг 4: Ваш сайт готов!
1. Подождите около 2-5 минут. GitHub нужно время на генерацию сайта.
2. Обновите страницу настроек **Pages**. В самом верху появится зеленая галочка и надпись:
   `Your site is live at https://ваш-никнейм.github.io/sigmavpn/`
> **Важно:** Не забудьте открыть файл `index.html` перед загрузкой (через Блокнот или VS Code) и заменить все ссылки `https://t.me/SIGMAVPN1BOT` на вашу, если нужно поменять бота! А также можете вставить коды Яндекс.Метрики и VK Реклама в места с надписью `<!-- Вставьте код... -->`.

## Шаг 5: Подключение вашего домена sigmavpn.space
Так как у вас уже есть крутой домен **sigmavpn.space**, давайте привяжем его:
1. Я уже создал для вас файл **CNAME** с текстом `sigmavpn.space` внутри папки. Он загрузится вместе с остальными файлами на GitHub.
2. В настройках **GitHub Pages** (раздел Settings -> Pages) найдите поле **Custom domain**.
3. Введите туда `sigmavpn.space` и нажмите **Save**. (GitHub может проверяться некоторое время).
4. Теперь зайдите в панель управления вашего регистратора (там, где вы купили домен sigmavpn.space) и найдите раздел **управления DNS-записями** (DNS Settings).
5. Создайте **четыре A-записи**, ведущие на IP-адреса GitHub:
   - Имя (Хост/Subdomain): `@` (или пусто) | Значение (IP): `185.199.108.153`
   - Имя (Хост/Subdomain): `@` (или пусто) | Значение (IP): `185.199.109.153`
   - Имя (Хост/Subdomain): `@` (или пусто) | Значение (IP): `185.199.110.153`
   - Имя (Хост/Subdomain): `@` (или пусто) | Значение (IP): `185.199.111.153`
6. (Дополнительно) Запись для www: Создайте **CNAME-запись**:
   - Имя (Хост): `www` | Тип: `CNAME` | Значение: `<ваш-github-никнейм>.github.io`
7. Сохраните изменения в регистраторе. Обновление DNS может занять от 15 минут до 24 часов (обычно быстро). Скоро ваш сайт будет доступен по адресу `https://sigmavpn.space`!
